# Online Chess Platform - User Stories

### Epic E1 - Authentication and Account Management

| US ID | Epic | Feature | Story | Priority |
|--------|------|---------|--------|----------|
| US-001 | E1 | Registration | As a user, I want to register with email/username so that I can create a chess account. | High |
| US-002 | E1 | Login | As a user, I want to log in securely so that I can access my games and rating. | High |
| US-003 | E1 | Session Continuity | As a user, I want token refresh so that I stay logged in during gameplay. | High |
| US-004 | E1 | Logout | As a user, I want to log out so that my account is secure on shared devices. | High |
| US-005 | E1 | Profile Management | As a user, I want to update my profile so that my chess identity is accurate. | Medium |
| US-006 | E1 | Password Recovery | As a user, I want password reset so that I can recover my account. | High |
| US-007 | E1 | Account Deactivation | As a user, I want to deactivate my account so that I can stop using the platform. | Low |
| US-008 | E1 | Admin Control | As an admin, I want user management controls so that I can handle abuse or violations. | Medium |
